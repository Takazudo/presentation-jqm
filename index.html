Components - Core

  • Core（必須）
    Base file for jQuery Mobile
  • Events（必須）
    Custom events and shortcuts.
  • Init（必須じゃないけど大体必要）
    Global initialization of the library.
  • Page Creation（必須じゃないけど大体必要）
    Basic page definition and formatting.
  • Page Sections
    Theming and layout of headers, footers, and content areas
  • Support Tests（必須）
    Assorted tests to qualify browsers by detecting features
  • Virtual Mouse (vmouse) Bindings（必須）
    Normalizes touch/mouse events.
  • Widget Factory（必須）
    Widget factory extentions for mobile.

Components - Forms

  • Buttons: Link-based
    Applies button styling to links
  • Controlgroups
    Visually groups sets of buttons, checks, radios, etc.
  • Fieldcontainers
    Styling to responsively position forms and labels based on screen width and add visual separation
  • Buttons: Input or button-based
    Custom-styled native input/buttons
  • Checkboxes & Radio Buttons
    Consistent styling for checkboxes/radio buttons.
  • Selects
    Consistent styling for native select menus. Tapping opens a native select menu.
  • Selects: Custom menus
    Extension to select menus to support menu styling, placeholder options, and multi-select features.
  • Slider
    Slider form widget
  • Text Inputs & Textareas
    Enhances and consistently styles text inputs.

Components - Navigation

  • Hashchange
    Hashchange event management for AJAX navigation system
  • AJAX Navigation System
    Applies the AJAX navigation system to links and forms to enable page transitions
  • Pushstate Support
    history.pushState support, layered on top of hashchange.

Components - Transitions

  • Transition Core
    Animated page change core logic and sequence handlers
  • Flip Transition
  • Flow Transition
  • Pop Transition
  • Slide Transition
  • Slidedown Transition
  • Slidefade Transition
  • Slideup Transition
  • Turn Transition

Components - Utilities

  • Degrade Inputs
    Changes input type to another after custom enhancements are made (ex. range > numberic).
  • Link Classes
    Adds classes to links.
  • matchMedia Polyfill
    A workaround for browsers without window.matchMedia
  • “nojs” Classes
    Adds class to make elements hidden to A grade browsers
  • Zoom Handling
    Utility methods for enabling and disabling user scaling (pinch zoom)
  • iOS Orientation Change Fix
    Fixes the orientation change bug in iOS when switching between landspace and portrait

Widgets

  • Collapsible
    Creates collapsible content blocks.
  • Collapsible Sets (Accordions)
    For creating grouped collapsible content areas.
  • Dialogs
    Displays a page as a modal dialog with inset appearance and overlay background
  • Toolbars: Fixed
    Behavior for “fixed” headers and footers
  • Grid Layouts (Columns)
    Applies classes for creating grid or column styling.
  • Listview
    Applies listview styling of various types (standard, numbered, split button, etc.)
  • Listview: Filter
    Extends the listview to add a search box to filter lists
  • Navbars
    Formats groups of links as horizontal navigation bars.

JavaScript

<script>
  // hashだったらhashナシにリダイレクト
  if(location.hash !== '') {
    location.href = location.href.replace(/#.*/,'');
  }
  // pushstateとajaxは使わない
  $.mobile.pushStateEnabled = false;
  $.mobile.ajaxEnabled = false;
</script>
